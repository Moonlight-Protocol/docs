# SDK

The Moonlight SDK gives developers the tools to interact with Moonlight privacy pools on Stellar. It handles key derivation, UTXO management, operation construction, and transaction signing.

Published on [JSR](https://jsr.io/@moonlight/moonlight-sdk) as `@moonlight/moonlight-sdk`. Current version: **0.7.0**.

Source: [github.com/Moonlight-Protocol/moonlight-sdk](https://github.com/Moonlight-Protocol/moonlight-sdk)
